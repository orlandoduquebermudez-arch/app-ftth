SISTEMA FTTH PWA

Funciones nuevas:
- Instalable como app en Android/Chrome.
- Funciona sin internet despues de abrirla una vez.
- Perfil de usuario: Administrador, Supervisor o Tecnico.
- Consecutivo automatico de orden.
- Captura de foto de evidencia.
- Firma del usuario en pantalla tactil.
- Captura GPS.
- Reporte individual por orden.
- Exportacion CSV.
- Respaldo e importacion JSON.

Uso:
1. Descomprimir el ZIP.
2. Abrir index.html en Chrome.
3. En celular, usar Instalar App si aparece.
4. Guardar respaldos JSON periodicamente.

Nota: Los datos se guardan localmente en el navegador. Para varios usuarios simultaneos se necesita backend con base de datos.
